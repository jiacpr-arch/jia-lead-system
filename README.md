# JIA Lead System 🏥

ระบบจัดการ Lead สำหรับ JIA TRAINER CENTER

## ฟีเจอร์
- 📂 แปลง CSV จาก Facebook Ads Manager อัตโนมัติ
- 📋 จัดการ Lead — สถานะ / ผู้รับผิดชอบ / หมายเหตุ
- 📊 Dashboard — กราฟสถานะ, ผลงานทีม, คอมมิชชั่น
- 📋 คัดลอกไปวาง Google Sheet ได้เลย

## วิธีใช้งาน
1. เปิดเว็บ → ลาก CSV จาก Facebook Ads Manager มาวาง
2. ตรวจสอบ + เลือกสถานะ/ผู้รับผิดชอบ
3. กด "คัดลอกไปวาง Sheet" → เปิด Google Sheet → คลิก A4 → Ctrl+V

## Deploy บน GitHub Pages
ไฟล์ `index.html` เดียว ไม่ต้อง build — deploy ได้ทันที
